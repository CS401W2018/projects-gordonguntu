CSS

Inline CSS is used within a html tag, embedded CSS is placed in the style tag inside the html file in head and external CSS is in a seperate file linked to html file where it is needed.

External CSS is preferred because it is easy to manage and even if we have multiple pages one externaL CSS can make changes for all of them.

A selector is used to choose specific html tag to apply style.
