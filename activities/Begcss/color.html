Tag name, Descendant rule, ID and Class are the four ways to select an item.

ID it is a unique identity used only to name one tag which usually can be a noun in stylesheet
It is to name more than one tag that specifies a class of a tag in the stylesheet.

Span tag is used to mark a part of a sentence in a paragraph and can allow that particular phrase to have an id or class stylesheet.
Div tag contains a divison or a section usually used to define them.
