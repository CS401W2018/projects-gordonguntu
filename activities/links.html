Absolute link and Relative link

Absolute link is a hyperlink that has a url for a page of other domain that leads to a different page

Relative link is a hyperlink that helps us to have the url of a file or page on the same website, so if we want to have a different page on the same domain we use relative link, it has a specific directory way.It follows the same path of the domain page.
