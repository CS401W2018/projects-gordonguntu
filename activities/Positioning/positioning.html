What is normal flow in HTML?

Normal flow is the default order where the elements appear in html.

What does float do? Why do you need to clear the float?

Float helps to move an element to position left or right side of the container. We need to clear float because if not they are removed from the document flow.

What do absolute, fixed, and relative do?

Position fixed makes the element stay there when scrolling the page.
Position absolute makes the element stay at a specific place.
Position relative makes the element stay in the normal place in the document flow.
